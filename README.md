Bu proje HTML, CSS ve JavaScript kullanarak Agar.IO oyununun basit bir kopyasını yapmak amacıyla oluşturuldu. Kullanıcıya basit bir arayüz ile kullanıcı adı ve hareket ettirecekleri karakterin(dairenin) rengini seçtirir ardından da oyunu oynamasını sağlar. Oyuncu oyun alanında bulduğu minik noktaları(yemleri) yiyerek kütlesini arttırır ve kendisinden küçük olan düşmanları yer. Bu süreçte büyük düşmanlar tarafından yenmemeye çalışır, aksi takdirde oyun mağlubiyetle sona erer.
